# Civonix — Test App

This is a **test/prototype build** of the Civonix app. It is not the final product.

## What's Working

- User authentication (login / register)
- Splash screen and navigation shell
- Basic UI screens: Dashboard, Portfolio, AI Insights, Leaderboard, News
- Theme and color system
- API client with token storage and auto-refresh

## What's Not Real / Incomplete

- Exchange API key connection is a UI placeholder — no real exchange data is being pulled
- Dashboard metrics and portfolio numbers are not live
- AI Insights are not connected to a real AI backend
- Leaderboard data is mocked or empty
- News feed may not load real articles
- Some screens show empty states because the backend is not running or fully built

## Notes

- The backend must be running locally at `http://192.168.1.3:8000` for API calls to work
- This app is for internal testing only — not for production use
- UI, flows, and features will change significantly in the real version
